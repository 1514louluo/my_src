typedef opt::v::dynamic_buffer<void *> cds::container::mspriority_queue::type_traits::buffer

Storage type.

The storage type for the heap array. Default is cds::opt::v::dynamic_buffer is used.

At MSPriorityQueue class instantiation, the buffer::rebind member metafunction is called to change type of values stored in the buffer. So, you may specify any type of values here.

typedef opt::none cds::container::mspriority_queue::type_traits::compare

Priority compare functor.

No default functor is provided. If the option is not specified, the less is used.

typedef opt::none cds::container::mspriority_queue::type_traits::less

specifies binary predicate used for priority comparing.

Default is std::less<T>.

typedef cds::opt::v::assignment_move_policy cds::container::mspriority_queue::type_traits::move_policy

Move policy.

The move policy used in MSPriorityQueue::pop functions to move item's value. Default is opt::v::assignment_move_policy.

typedef cds::opt::v::default_swap_policy cds::container::mspriority_queue::type_traits::swap_policy

The swap policy is used in heapify algorithm. Default is opt::v::default_swap_policy.
